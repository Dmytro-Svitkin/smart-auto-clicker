# smart-auto-clicker

Smart Auto Clicker is a lightweight tool that automatically assists with rapid clicking.

## Description
- Detects when the user is clicking very fast.
- Automatically switches into autoclicker mode when rapid clicks are detected.
- Autoclicker mode stops automatically once clicking slows down.
- Early version: runs in console mode, no GUI available yet.
- To terminate the program, close the console window or press and hold `CTRL+C`.

## Usage
- Run the program.
- Begin clicking normally; if rapid clicking is detected, autoclicker mode will activate.
- Autoclicker mode will stop automatically when clicking slows down.
- To exit the program:
  - Press and hold `CTRL+C`. 
  - Close the console window.
  
## Disclaimer
- Currently compatible only with Windows 10 and Windows 11.
- `CTRL+C` termination causes an error, this is expected and does not affect functionality.
